# Telegram Films Bot

#  Опис
Films Bot — це Telegram‑бот для роботи з базою фільмів. Він дозволяє:
- додавати нові фільми,
- переглядати список,
- шукати за актором,
- фільтрувати за жанром чи рейтингом,
- редагувати та видаляти записи.  

Дані зберігаються у файлі `data.json`.
