# MovieTelegramBot 🤖🍿

Інтелектуальний Telegram-бот на Python для пошуку, підбору та рекомендації фільмів за запитами користувачів.

## 🛠️ Функціонал
* **Пошук фільмів:** Інтерактивне меню для миттєвого пошуку інформації про кінорелізи.
* **Інтеграція з API:** Отримання актуальних даних (описи, рейтинги, постери) з кінобаз у реальному часі.
* **База даних користувачів:** Збереження уподобань користувачів та історії їхніх запитів для покращення рекомендацій.
* **Оптимізована архітектура:** Асинхронна обробка запитів та зручна система кастомних клавіатур для навігації.

## 🚀 Стек технологій
* Python (aiogram / telebot)
* API Інтеграції (Requests / HTTPX)
* SQLite3 (збереження логів та налаштувань користувачів)
